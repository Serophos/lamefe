WICHTIGE HINWEISE ZU DIESEM RELEASE:
==============================================================
· Achtung seit LameFE 2.2 Beta 6 hat sich das Format der Konfi-
  gurationsdatei geändert. Bitte verwenden Sie NICHT Ihre alte
  Version der Konfigurationsdatei (LameFE.ini) weiter.
· Wenn Sie das Preset-Verzeichnis von LameFE ändern, kopieren
  Sie bitte erst die Presets in das neue Verzeichnis, bevor
  Sie LameFE neustarten.
· Wenn Sie die Highcolor Icons unter Windows 98/ME benutzen
  möchten, müssen Sie die neueste Version der Common Controls
  Bibliothek installiert haben. Bitte führen Sie Windows Update
  aus um diese zu installieren oder laden Sie das Update von
  www.microsoft.de runter.
· Auf einigen Systemen benötigen Sie einen ASPI Treiber um
  LameFE benutzen zu können. Sollten Probleme beim verwenden
  von LameFE auftreten (Ausnahmefehler in CDRip.dll etc.) 
  sollten Sie als ersten Lösungsansatz neue ASPI Treiber
  installieren. Sie können diese entweder beim Hersteller 
  Ihres CD-Laufwerkes oder von lamefe.sourceforge.net runter-
  laden.
· ACHTUNG. Diese Version von LameFE ist eine Beta-Version.
  Auch wenn die meisten Programmfunktionen fehlerfrei funktio-
  nieren, können doch unterwartete Fehlfunktionen auftreten,
  die auch die Systemstabilität beeinflussen können. Sie be-
  nutzen dieses Programm auf eigene Gefahr (siehe auch GPL)!
